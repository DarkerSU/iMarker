拖拽生成html代码 ​

要拖动元素并生成 HTML 代码，您需要做以下几个步骤：

  1. 创建要拖动的 HTML 元素，并将其设置为可拖动。您可以使用 draggable 属性来实现此操作。
HTML
<div class="drag-element" draggable="true">拖动我生成 HTML 代码</div>
  1. 添加指定元素的 dragstart 事件的监听器，以在拖动时获取该元素的数据并设置数据类型，如下所示：
javascript
const dragElement = document.querySelector('.drag-element');

dragElement.addEventListener('dragstart', event => {
  event.dataTransfer.setData('text/plain', dragElement.outerHTML);
  event.dataTransfer.dropEffect = 'move';
});
  1. 在放置区域的 drop 事件中获取该数据，并在用户放下元素时将其插入 HTML 中。
javascript
const dropZone = document.querySelector('.drop-zone');

dropZone.addEventListener('drop', event => {
  event.preventDefault();
  const code = event.dataTransfer.getData('text/plain');
  const element = document.createElement('div');
  element.innerHTML = code;
  dropZone.appendChild(element.childNodes[0]);
});

dropZone.addEventListener('dragover', event => {
  event.preventDefault();
  event.dataTransfer.dropEffect = 'move';
});

这里用一个类名为 .drag-element 的元素作为拖拽源，并且在添加 dragstart 的事件监听器时获取该元素的 HTML 代码。当放置区域的 drop 事件触发时，将获取到的 HTML 代码转成 HTML 元素并将其添加到放置区域的内部。在放置区域的 dragover 事件中，防止浏览器默认处理拖放操作并控制鼠标光标的外观。

通过以上这些操作，您就可以通过拖动元素来生成 HTML 代码啦。如果您需要更多的自定义选项和复杂操作，您可以使用第三方拖拽库，例如之前提到的 SortableJS 或 draggable 等插件，可以更方便地实现拖拽及其相关功能。
